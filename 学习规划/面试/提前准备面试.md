# 职业规划

在b站评论区看到：计算机行业真的有中年危机：未来计算机行业人越来越多，与其把钱浪费在一个低效且工资高的中年人身上，不如找一个年轻人，或许只要培养半年，就完全足够代替中年人。
所以有两个选择：

1. 转行：那么就要多注意别的行业、多结识人。创业、投资
2. 提升自身技能，即使再提升也会有一天思维跟不上。老的人不一定经验少（老的科学家、邓小平），但是反应一定慢了。这时候怎么学习新技术？
3. 进入管理层
